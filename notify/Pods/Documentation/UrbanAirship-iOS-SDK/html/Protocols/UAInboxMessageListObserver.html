UAInboxMessageListObserver Protocol Reference

Conforms to NSObject
Declared in UAInboxMessageListObserver.h

Overview

The UAInboxMessageListObserver protocol declares methods that are implemented by observers of a UAInboxMessageList object.

An observer is notified when messages are modified, loaded or fail to load or update.

Tasks

  • – messageListWillLoad

    Tells an observer when a message list reload or update is initiated.
  • – messageListLoaded

    Tells an observer when the message list has successfully loaded the latest information from Urban Airship’s servers.
  • – inboxLoadFailed

    Tells the observer that a request for inbox messages failed.
  • – singleMessageMarkAsReadFinished:

    Tells the observer that a message has been marked as read.
  • – singleMessageMarkAsReadFailed:

    Tells the observer that a mark-as-read request failed.
  • – batchMarkAsReadFinished

    Tells the observer that a batch of messages has been marked as read.
  • – batchMarkAsReadFailed

    Tells the observer that a batch mark-as-read request failed.
  • – batchDeleteFinished

    Tells the observer when a batch of messages has been deleted.
  • – batchDeleteFailed

    Tells the observer when a batch update of messages has failed.
