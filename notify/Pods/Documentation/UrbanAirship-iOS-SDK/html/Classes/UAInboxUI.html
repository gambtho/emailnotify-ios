UAInboxUI Class Reference

Inherits from NSObject
Conforms to UAInboxPushHandlerDelegate
UAInboxUIProtocol
Declared in UAInboxUI.h

Overview

This class is the default rich push UI impelementation. When it is designated as the [UAInbox uiClass], launching the inbox will cause it to be displayed in a modal view controller.
