purchasedProducts

Array of all products purchased by this user.

@property (nonatomic, retain, readonly) NSMutableArray *purchasedProducts

Discussion

Array of all products purchased by this user.

Note that these products are copies of the products available through the products property. Only these copies have the [UASubscriptionProduct purchased] flag set and valid [UASubscriptionProduct startDate] and [UASubscriptionProduct endDate] values.
